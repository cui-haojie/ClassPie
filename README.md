# ClassPi 课堂派课程设计说明

## 项目结构

- 后端：`ClassPiServer`（Spring Boot + MyBatis，端口 9090）
- 前端：`classPai`（Vue 3 + Vite，端口 5174）

## 启动前准备

1. 启动 MySQL，确保存在数据库 `t_class`
2. **执行一次**数据库升级脚本：

```sql
-- 文件位置：src/main/resources/schema_update.sql
```

3. 启动后端 `ClassPiServerApplication`
4. 启动前端：

```bash
cd classPai
npm run dev
```
